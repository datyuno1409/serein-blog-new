⚠️ API Không Khả Dụng

Bạn đang cố gắng truy cập vào API endpoints của website.

Tuy nhiên, Netlify không hỗ trợ PHP backend nên các API endpoints không thể hoạt động.

API hiện tại sử dụng:

• PHP với MySQL database

• RESTful API endpoints

• Session management

Để sử dụng API:

1. Deploy lên server hỗ trợ PHP

2. Sử dụng Netlify Functions

3. Chuyển sang Node.js API

4. Sử dụng Serverless functions
