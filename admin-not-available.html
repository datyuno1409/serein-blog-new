🚫 Admin Area Không Khả Dụng

Xin chào! Bạn đang cố gắng truy cập vào khu vực quản trị của website.

Tuy nhiên, Netlify chỉ hỗ trợ hosting static files (HTML, CSS, JS) và không thể chạy các ứng dụng PHP với database.

Tại sao không thể truy cập?

• Admin area sử dụng PHP và MySQL database

• Netlify chỉ hỗ trợ static hosting

• Cần server hỗ trợ PHP như VPS, Shared hosting, hoặc Heroku

Giải pháp:

1. Sử dụng VPS/Cloud server với LAMP/LEMP stack

2. Hosting PHP như 000webhost, InfinityFree

3. Chuyển sang Headless CMS như Strapi, Sanity

4. Sử dụng Netlify Functions với serverless backend
